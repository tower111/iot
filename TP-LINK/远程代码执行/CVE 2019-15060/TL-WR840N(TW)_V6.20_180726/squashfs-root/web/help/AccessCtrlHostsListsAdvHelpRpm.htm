Add or Edit a Host Entry Help

You can create an entry for the host list on this page.

  • Mode - Here are two options, IP Address and MAC Address. You can select either of them from the drop-down list.
  • Description - In this field, create a unique description for the host.

If the IP Address is selected, you can see the following item:

  • LAN IP Address - Enter the IP address or address range of the host in dotted-decimal format, for example 192.168.0.23.

If the MAC Address is selected, you can see the following item:

  • MAC Address - Enter the MAC address of the host in XX:XX:XX:XX:XX:XX format, for example 00:11:22:33:44:AA.

Click Save to complete the settings.

Click Back to return to the Host Settings page.
